# Projeto Portas Abertas

### 1. Descrição do Projeto

Este projeto está sendo desenvolvido para a disciplina "ES46F/ES61 - Oficina de Integração 1" da UTFPR. O objetivo é criar uma solução digital que centralize, organize e facilite o acesso a informações sobre instituições e serviços voltados à neurodiversidade. A plataforma atua como um portfólio de instituições dedicadas ao atendimento de pessoas neurodivergentes e com deficiência (PCDs).
